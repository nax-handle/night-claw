# 🌙nightclaw

Simple AI agent style CLI.

## Run

```bash
npm start
```

First-time setup:

```bash
nightclaw setup
```

Dev mode (TypeScript, no build output):

```bash
npm run dev
```

Single prompt mode:

```bash
npm start -- "build login api"
```

Help:

```bash
npm start -- --help
```

Typos alias supported:

```bash
nigthclaw --help
```

Planning notes live in `plan/cli-plan.md`.
